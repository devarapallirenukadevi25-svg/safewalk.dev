# Safe Walk
This is an Android app ,an inclusive personal safety application designed for everyone вЂ” including men, women, students, elderly individuals, and specially-abled users such as the deaf, mute, and visually impaired. which Will Call to an Emergency number, based on a specific voice command like "Help" .
